Modern Typing Guide

Two Ways to Type Arabic

Modern Mode (Recommended)

Type naturally with intuitive letter combinations. Perfect for beginners and mobile users. Numbers automatically convert to Arabic letters for dialectal writing.

  • Type sh for ش, th for ث, kh for خ
  • Numbers convert automatically: 3 → ع, 7 → ح
  • Natural flow, no special characters needed
  • Works great on mobile keyboards

Legacy Mode

Compatible with traditional Arabic keyboard systems. Uses apostrophes and capitals for special characters. Perfect for users migrating from other tools.

  • Type apostrophe for variants: s' → ش, t' → ث
  • Use capitals for emphatic letters: H → ح, S → ص
  • Type = after numbers for Eastern Arabic numerals
  • Traditional system, familiar to many users

Choose What Works For You

Switch between modes anytime with the toggle above. Your preference is automatically saved. Both modes produce perfect Arabic text - use whichever feels more natural to you.
